# my-portfolio
# Интерактивное консольное портфолио

Мой личный проект-портфолио на Python. Приложение запускается в консоли, а перед стартом выводит графическое анимированное интро.

## Инструкция по запуску

1. Скачайте все файлы проекта в одну папку.
2. Откройте терминал в этой папке и запустите главный файл командой:
python Portfolio11.py

## Как пользоваться

1. После запуска откроется графическое окно с анимацией логотипа (оно закроется само через секунду).
2. В консоли появится меню. Введите цифру от 1 до 8, чтобы открыть нужный раздел.
3. Чтобы вернуться обратно в меню после чтения раздела, нажмите Enter.
4. Для выхода из программы введите 0.
