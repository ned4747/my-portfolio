1) Проект "kamen.py". Данный код создавался для проектной работы 1. Ссылка: https://github.com/ned4747/my-portfolio/blob/main/kamen.py
В этой программе пользователь играет с ботом в камень-ножницы-бумага. После каждой победы/поражения в консоль выводится счет. Сама игра идет до того момента, пока один из двух игроков не выиграет 3 раза. Выбор бота зависит от случайного выбора одного из трех объектов. В конце выводится конечный счет.
2) Проект "own graphics.py". Данный код был создан для практики и для работы с библиотекой tkinter (Написан 2-3 месяца назад). Ссылка: https://github.com/ned4747/my-portfolio/blob/main/own%20graphics.py
С помощью данного кода можно смотреть свои статусы/ранги в разных играх (На данный момент доступен фиксированный выбор из двух игр) и изменять их. При запуске программа запрашивает пароль. После правильного ввода кода открывается главное меню, где находятся сами игры и их статусы. Данные сохраняются в виде текстового файла. Пароль — "krutoi parol".
